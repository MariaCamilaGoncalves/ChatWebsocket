💬 Chat em Tempo Real
Chat em tempo real feito com Java + Spring Boot e WebSocket.

## Tecnologias
- **Java 17**
- **Spring Boot 4.0.3**
- **WebSocket + STOMP**
- **SockJS**
- **HTML, CSS e JavaScript**

## ⚙️ Como rodar
### Pré-requisitos
- Java 17+
- Maven

### Passos
1. Clone o repositório:
```bash
git clone https://github.com/MariaCamilaGoncalves/ChatWebsocket.git
cd chat
```
2. Rode o projeto:
```bash
./mvnw spring-boot:run
```
3. Acesse no navegador em duas abas para testar o chat entre dois usuários!
```
http://localhost:8080
```

## 💡 Funcionalidades
- Conexão em tempo real via WebSocket
- Múltiplos usuários simultâneos
- Mensagem de entrada quando um usuário entra no chat
- Seletor de emojis
- Interface responsiva com tema escuro
